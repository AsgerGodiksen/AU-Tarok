# AU - Tarok
This project aims to build a new quadruped for Aarhus University insprided by the biomehanics of a Horse.

Aarhus University have previous designed and manufactured a quadruped robot and this project aim to build on the lessons leared from that project.

Where the previous mistakes will be fixed and other design choises is taken for the developmend of a new and better quadruped based on the locomotion of a Horse. 

This page main goal is ensure a better knowlage sharing for this project, based on the lack of same from the earlier project with the expisting quadruped
